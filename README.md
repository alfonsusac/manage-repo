# manage-repo-config

`manage-repo-config` is a command-line tool for inspecting local repository configuration by running a local web server which allow user to configure their project in a nicer UI.

## Installation

```bash
bunx manage-repo
npx manage-repo
pnpx manage-repo
```

### Options

- `--port` allows you to change the port used for the app.
- `--host` allows you to change the host used for the app.
- `--showdir` shows the directory path of where the command is run
- `--no-analytics` disable analytics tracking
